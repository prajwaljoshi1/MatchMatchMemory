# Match Match Memory

======================================================================================================================

jQuery example app that implements the famous memory game of finding matching pairs of letters. A good brain exercise.

### click [here][1] to play Match Match Memory


## Libraries Used.

- [Sweet Alert][2]
- [Firebase][3]
- [Animated Model][4]
- [Facebook Javascript SDK][5]
